## 关键设计决策
- 采用单文件 HTML/CSS/JS 方案，无需构建工具，方便直接打开和二次修改。
- 使用 `bg1.jpeg` 作为全屏背景，配合 CSS 暗角遮罩、噪点纹理、漂浮光球，增强画面层次与科技感。
- 标题使用 `background-clip: text` 实现银白渐变，与 PPT 中 `Physical AI Hackathon` 文字质感一致。
- 卡片与按钮统一使用 `backdrop-filter` 玻璃拟态，呼应背景中的半透明气泡。
- 开场动画使用有限时间序列（logo 呼吸、扫描线、进度条、术语切换），避免无限循环动效杂乱。
- 问卷以 overlay 方式接入，不破坏首页视觉完整性，支持返回首页。
- 使用 SVG 图标替代 emoji，符合专业 UI 规范。
- 加入 `prefers-reduced-motion` 媒体查询，支持可访问性。
- 结果页根据用户选择动态生成画像类型，增加仪式感和分享欲。
